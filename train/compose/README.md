Композиция классов используется как альтернатива насдедования. Частьв случаюх, когда статическиое определение не удобно.

Композиция объектов выполняется динамически во время выполнения программы.

Главный механизм: Объекты получают ссылки на другие объекты.
Композиция позволяет строго разделить задачи между классами, а также сделать структуру более минималистичной, чем при использовании наслеования.
