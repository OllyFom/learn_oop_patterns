## Что это за место? 

Этот репозиторий существует как единое место шаблонов и проб для паттернов ООП на C++

В папке *train* расположены основы ооп, которые лежат в основе паттернов: композиция, инверсия управления и делегирование

Наследование не рассмотрено, но ему тоже тут место

В папке *patterns* расположены сами простейшие примеры паттернов. Они не разделены по классовым и объектным паттернам явно, но 
об этом есть пометки внутри, в файлах README.md
